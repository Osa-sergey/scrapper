## Сгенерировал небольшой роадмап для веб разработки

Запрос:

```
Создай минималистичный roadmap по фронтенд-разработке в формате HTML

Требования:
- Древовидный список

- Для описания одного элемента
   - Название
   - Описание (2-3 предложения на русском)  
   - Ключевые слова (5-10 конкретных терминов через запятую)

- Основные разделы
Базовые технологии, Фреймворки, Инструменты разработки

- Стиль
Без лишних элементов, только текст  
```
